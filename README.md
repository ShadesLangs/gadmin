# G<ins>admin</ins>
Free Bootstrap 4 Gentelella inspired admin template

![Preview 1](https://github.com/christianesperar/gadmin/blob/master/preview.jpg)


## Installation

**npm**

```
npm install gadmin --save
```

**yarn**

```
yarn add gadmin --save
```
